# クイックスタート - 実戦での使い方

## 今すぐ予測する

### 方法1: Webアプリ（推奨）
```bash
python web_app.py
```
→ http://localhost:8000 を開く

**必要な情報（KEIRIN.JPの出走表から取得）:**
1. 各選手の**競走得点** (例: 115.20)
2. 各選手の**級班** (SS/S1/S2/A1/A2/A3)
3. 各選手の**脚質** (逃/追/両)
4. 各選手の**登録地** (都道府県)

入力すると→高配当確率が表示される（例: 45.2%）

---

### 方法2: コマンドライン
```bash
python easy_predict.py --date 20241025 --min-score 0.4
```

過去データのOOF予測から、確率40%以上のレースを表示

---

## 実際のレースで使う手順

### ステップ1: KEIRIN.JPで出走表を開く
例: https://keirin.jp → 開催中のレース → 出走表

### ステップ2: 必要なデータを取得

各選手について:
```
車番  選手名    競走得点  級班  脚質  登録地
1    山田太郎   115.20   S1    逃    埼玉
2    鈴木次郎   113.45   S1    追    大阪
...
```

### ステップ3: Webアプリに入力

1. `python web_app.py` を起動
2. ブラウザで http://localhost:8000
3. データを入力
4. 「予測する」クリック

### ステップ4: 結果を見る

```
高配当確率: 42.3%
予測: MEDIUM-HIGH (やや荒れる)
期待配当: ¥10,000 - ¥20,000
```

## 確率の読み方

| 確率 | ラベル | 期待配当 | 投資判断 |
|------|--------|----------|----------|
| 40%以上 | HIGH | ¥15,000以上 | 高配当狙い |
| 30-40% | MEDIUM-HIGH | ¥10,000-20,000 | 中穴狙い |
| 20-30% | MEDIUM | ¥5,000-12,000 | 普通 |
| 20%未満 | LOW | ¥8,000未満 | 本命決着 |

**注意**: 確率が高い = 荒れる可能性が高い = リスクも高い

## トラブルシューティング

### Q: 競走得点がわからない
A: KEIRIN.JP出走表の「4ヶ月評価」欄を見る

### Q: 脚質がわからない  
A: 選手プロフィールで確認、または逃げ=先行型、追=差し型

### Q: 予測が外れた
A: システムは確率を出すだけ。100%的中はありません。
   長期的に高確率レースを狙うことで利益を出す戦略です。
